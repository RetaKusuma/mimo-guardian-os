# MiMo Guardian OS

**Autonomous AI Operating Layer for Xiaomi Ecosystem**

MiMo Guardian OS is a multi-agent AI control center powered by Xiaomi MiMo. It transforms a Xiaomi device into a self-optimizing personal operating layer: predicting battery drain, cleaning storage, summarizing notifications, searching memory, validating safe actions, and executing automation workflows.

## Why this project is different

This is not just a phone cleaner. It is an AI agent runtime for Xiaomi devices:

- Planner Agent decomposes user intent
- Device Agent monitors storage, RAM, thermal, and battery
- Notification Agent summarizes and prioritizes alerts
- Memory Agent remembers context and usage patterns
- Search Agent finds screenshots, files, and media
- Validator Agent checks action safety
- Automation Agent executes approved actions

## Demo Flow

1. User enters: `Optimize my phone for tonight`
2. Planner Agent creates subtasks
3. Specialist agents scan device signals
4. Validator approves safe actions
5. Automation Agent executes cleanup, smart charging, and focus mode
6. Dashboard updates metrics and runtime logs
